# Paddy Seed Classifier API Documentation

This API provides a deep learning-based inference service to classify the quality of Paddy Seeds into two categories: **Pure** and **Impure**.

## Base URL
The base URL is generated by Google Cloud Run after deployment. It will look like: 
`https://resnet-api-297419987783.asia-south1.run.app`

---

## Endpoints

### 1. Health Check
Checks if the API service is running correctly.

*   **URL:** `/`
*   **Method:** `GET`
*   **Response:**
    ```json
    {
      "message": "Welcome to the Paddy Seed Classifier API. Use /predict to classify seeds."
    }
    ```

---

### 2. Predict Seed Quality
Uploads an image of a paddy seed and returns the classification result.

*   **URL:** `/predict`
*   **Method:** `POST`
*   **Content-Type:** `multipart/form-data`
*   **Request Body:**
    | Parameter | Type | Description | Required |
    | :--- | :--- | :--- | :--- |
    | `file` | `File (Binary)` | The image of the paddy seed (JPG, PNG, JPEG). | Yes |

*   **Success Response (200 OK):**
    ```json
    {
      "filename": "seed_sample.jpg",
      "prediction": "pure",
      "confidence": 0.9842
    }
    ```

*   **Error Responses:**
    *   **400 Bad Request**: If the uploaded file is not an image.
    *   **500 Internal Server Error**: If there is an issue during model inference.

---

## Technical Details
*   **Model Architecture:** ResNet50 (Transfer Learning)
*   **Input Requirements:** 
    *   Images are automatically resized to **224x224** pixels.
    *   Normalization is applied using ImageNet mean `[0.485, 0.456, 0.406]` and std `[0.229, 0.224, 0.225]`.
*   **Categories:** `pure`, `impure`.

---

## Example Usage (cURL)

```bash
curl -X POST "https://resnet-api-297419987783.asia-south1.run.app/predict" \
     -H "accept: application/json" \
     -H "Content-Type: multipart/form-data" \
     -F "file=@/path/to/your/seed_image.jpg"
```
